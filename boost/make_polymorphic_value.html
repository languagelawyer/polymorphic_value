Function template make_polymorphic_value

boost::make_polymorphic_value

Synopsis

// In header: <boost/polymorphic_value.hpp>


template<typename T, typename U = T, class... Ts> 
  polymorphic_value< T > make_polymorphic_value(Ts &&... ts);

Description

Returns: A polymorphic_value<T> owning an object initialised with U(std::forward<Ts>(ts)...).
